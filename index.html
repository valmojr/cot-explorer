What is this?

This tool helps you understand Cursor on Target (COT) types. COT types are standardized, hierarchical identifiers used to classify military units, equipment, and activities in situational awareness systems. Enter a COT code below (like "a-f-A-M-F-C-H") and we'll show you all matching definitions.

Created by the folks at Provisor, the simpler ATAK for event organizers. Feedback & suggestions welcome at info@provisor.app.
