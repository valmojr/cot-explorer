What is this?

This tool helps you understand military Course of Action Type (COT) codes. COT codes are standardized identifiers used to classify military units, equipment, and activities. Enter a COT code below (like "a-f-A-M-F-C-H") and we'll show you all matching definitions from the official COT types database.
